# 极乐迪斯科人格测评器

仿照 SBTI（Silly Big Type Indicator）玩法，基于《极乐迪斯科》（Disco Elysium）24 种技能人格声音制作的娱乐向人格测评网站。

## 这是什么

《极乐迪斯科》中，主角哈里·杜博阿的 24 项技能并非冰冷的数值，而是 24 个在你脑海中争吵、低语、给出建议的独立人格声音。每项技能都有自己独特的语气、偏好和偏见。

本项目将这套体系做成了一个互动测试：

- **24 道情景题** —— 每题对应一项技能，用极乐迪斯科式的文学化语言描述场景
- **12 维子维度评分** —— 每两项技能合成一个子维度，压缩为 L/M/H 三档
- **16 种人格原型** —— 用曼哈顿距离将你的 12 维向量匹配到最接近的原型
- **24 人格声音图鉴** —— 浏览全部技能的详细介绍、高低等级表现和经典台词

## 技术栈

纯静态前端页面，零框架依赖。HTML + CSS + JavaScript，通过 GitHub Pages 部署。
